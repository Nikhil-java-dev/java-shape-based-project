# Java Shape Based Project

## Description
This project demonstrates core Java concepts such as classes, objects, constructors, encapsulation, and mathematical calculations using shape-based examples.

## Features
- Calculate area and perimeter of shapes
- Object-oriented design
- Clean and readable code

## Technologies Used
- Java
- OOP Concepts

## How to Run
1. Clone the repository
2. Open the project in Eclipse/IntelliJ
3. Run the main class
